# Лабораторная работа: Риэлторское агентство

## Описание проекта
Реализация объектной модели данных и unit-тестов для риэлторского агентства в рамках учебной лабораторной работы по разработке корпоративных приложений.

**Предметная область:** система учета заявок на покупку и продажу недвижимости.

## Структура решения
### RealEstateAgency.Domain
Библиотека с доменной моделью и бизнес-логикой.

#### Enums/PropertyPurpose.cs
**Назначение:** классификация недвижимости по целевому использованию.
#### Enums/PropertyType.cs
**Назначение:** классификация по физическим характеристикам объекта.
#### Enums/RequestType.cs
**Назначение:** определение направления операции с недвижимостью.

#### Models/Counterparty.cs
**Назначение:** представление клиента агентства (покупателя или продавца).
#### Models/RealEstateProperty.cs
**Назначение:** описание объекта недвижимости со всеми техническими характеристиками.
#### Models/Request.cs
**Назначение:** представление заявки как контракта между контрагентом и объектом недвижимости.

### RealEstateAgency.Tests
Библиотека модульных тестов для проверки бизнес-логики.

#### RealEstateTestFixture.cs
**Назначение:** генератор тестовых данных
#### RealEstateQueriesTests.cs
**Назначение:** модульные тесты для проверки корректности LINQ-запросов.
